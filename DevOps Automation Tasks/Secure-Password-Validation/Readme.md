Interview Task:
Write a script to validate a password based on the following enhanced criteria:

1. The password must be at least 10 characters long.


2. It should contain at least one uppercase, one lowercase, and one numeric digit.


3. It must include at least one special character (e.g., @, #, $, %).


4. The password should not contain consecutive repeating characters (e.g., "aa", "11").


5. If the password fails validation, output the specific reason.


6. Once validated, hash the password using a secure algorithm (e.g., bcrypt) before storing it.
